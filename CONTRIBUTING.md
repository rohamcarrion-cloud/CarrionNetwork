# Contributing to Carrion Networks

Thank you for your interest in Carrion Networks.

## Current Development Status

Carrion Networks is currently under active early-stage development. The project
is not yet accepting unsolicited feature contributions.

Bug reports, documentation corrections, and security reports are welcome.

## Development Setup

1. Clone the repository.
2. Install Node.js using the version in `.nvmrc`.
3. Run `npm install`.
4. Copy `.env.example` to `.env`.
5. Start the development environment with `npm run dev`.

## Quality Checks

Before submitting changes, run:

```bash
npm run lint
npm run build
