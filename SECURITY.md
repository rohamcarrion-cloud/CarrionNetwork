# Security Policy

## Supported Versions

Carrion Networks is currently in pre-release development. Security updates are
applied to the latest version on the `main` branch.

## Reporting a Vulnerability

Do not disclose suspected vulnerabilities through a public GitHub issue.

Please report security concerns privately to the project owner with:

- A description of the issue
- Steps to reproduce it
- The affected component
- Potential impact
- Any suggested remediation

Please avoid accessing, modifying, or downloading data that does not belong to
you while investigating a possible vulnerability.

## Current Security Scope

Security-sensitive areas include:

- Authentication
- Role-based authorization
- PocketBase collection rules
- File uploads
- Environment variables
- Administrative access
- User and creator data
